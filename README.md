# assess

A Claude Code plugin from Family IT Guy. Critique how you're directing the model: hold your prompt (or the session so far) up to the question-method lenses and sharpen it, instead of executing it. Summon with `/assess`.

## Install

`assess` ships in the Family IT Guy plugin marketplace. In Claude Code:

```
/plugin marketplace add Family-IT-Guy/plugins
/plugin install assess@familyitguy
```

Then type `/assess` (optionally append it to a draft prompt).

## Updates

To get new versions automatically: open the `/plugin` menu → **Marketplaces** → select `familyitguy` → toggle **auto-update on**. Or update by hand anytime:

```
/plugin marketplace update familyitguy
/plugin install assess@familyitguy
```
